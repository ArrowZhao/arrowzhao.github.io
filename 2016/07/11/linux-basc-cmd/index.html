Linux基本命令

2016-07-11 • 标签: Linux
1.上传下载文件工具rz和sz

yum install -y lrzsz
注意:rhel安装完系统后 直接可使用 rz sz
使用方法：
1）、上传文件：

rz 本地文件名
rz -y 本地文件名 覆盖上传
2）、下载文件：

sz remotefilename
sz -y remotefilename 覆盖下载
下载的默认路径在：windows的下载目录、在SecureCRT下执行的话在安装目录的download目录下。

2. 常用命令总结

1）、创建文件夹 mkdir
-p 递归删除子目录
2）、创建文件 touch

3）、删除文件夹 rm -rf

4）、删除文件 rm -f

5）、拷贝文件 cp 原路径 目标路径

6）、拷贝文件夹 cp -ri 原路径 目标路径

7）、查看文件内容 cat

8）、将着文本直接写入文件的末尾一行 catecho ‘i am a teacher’ >> oldboy.txt

9）、剪切 mv 原路径 目标路径

10）、head -行数 文件名 查看某个文件的前几行

11）、tail -行数 文件名 查看某个文件的结尾几行

12）、grep 核心命名 (三剑客老三) 过滤
-v 排除
-E 多个 竖线分隔 等于命令 egrep
-i 忽略大小写

13）、sed 核心命名 (三剑客老二) 主要是以行为单位进行处理，可以将数据行进行替换、删除、新增、选取等特定工作
详细用法 参见：http://www.cnblogs.com/dong008259/archive/2011/12/07/2279897.html
sed -n 20,30p ett.txt 输出文件的 20- 30行
sed -i s#1#2#g ett.txt 将1替换为2

14）、波浪线代表用户的目录 ~

15）、which 命令 查看命令的位置

16）、alias -p key=value 别名命令
unalias 取消别名
别名存储位置 ~\ .bashrc 文件中

17）、find 利用管道删除
find / -name ‘test.txt’ | xargs rm -rf 说明 xargs 安行
-type 类型 f文件
-name 文件名
-mtime 时间

18）、seq 打印序列
seq 100 打印 1- 100
seq 20 100 打印 20-100
seq 1 2 9 打印 1-9 以 2递增
seq -s ‘,’ 1 9 打印 1-9横向以逗号分隔

19）、chkconfig –list 查看运行的服务
chkconfig shhd off 设置shhd开启不启动

20）、设置时间同步 rdate -s time-b.nist.gov
